# Aura Intense

Sitio web del catalogo Aura Intense, preparado para desplegarse en GitHub Pages con Vite.

## Publicar en GitHub Pages

1. Subi esta carpeta a un repositorio nuevo en GitHub.
2. En GitHub entra a `Settings > Pages`.
3. En `Source`, elegi `GitHub Actions`.
4. Hace push a la rama principal para que se publique solo.

El workflow de deploy esta en `.github/workflows/deploy.yml`.

## Desarrollo local

```bash
npm install
npm run dev
```

## Build local

```bash
npm run build
```
